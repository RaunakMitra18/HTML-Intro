Way to become a Fullstack Web Developer

Languages

  1. HTML
  2. CSS
  3. JavaScript
  4. FrontEnd
    • React
    • Angular
    • Vue
    • Svelte
  5. BackEnd
    • Node
    • Django
    • Ruby on Rails
    • Laravel
  6. You DID NOT learn PHP
  7. Database SQL/NoSQL

Practices and Projects

Step 1

Start by learning basic HTML, CSS. Make a simple static website to test your knowledge.

Step 2

Start learning JavaScript and implement some logic behind your static site.

Step 3

Learn a frontend framework like React or Angular and start building a fairly large static project.

Step 4

After building the static website with React (or others), start with Node (or any other backend framework)

Step 5

Connect the frontend to the backend and start using Databases and Servers.

Step 6

Build a complete dynamic website with frontend and backend.

Step 7

Congrats, you are a fullstack developer!
